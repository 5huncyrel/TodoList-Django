## Backend API Endpoints


### 1. Fetch All Tasks
- **URL**: `/api/todos/fetch/`
- **Method**: `GET`

### 2. Fetch a single task by ID
- **URL**: `api/todos/<int:pk>/fetch/`
- **Method**: `GET`

### 3. Create a Task
- **URL**: `api/todos/create/`
- **Method**: `POST`

### 4. Update a task by ID
- **URL**: `api/todos/<int:pk>/update/`
- **Method**: `PUT`

### 5. DELETE a task by ID
- **URL**: `api/todos/<int:pk>/delete/`
- **Method**: `DELETE`
